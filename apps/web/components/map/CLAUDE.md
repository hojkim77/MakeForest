# Map — 맵 시스템 명세 (D)

## 숲 모드 (줌아웃)

- 시/구 단위 누적 생명체 픽셀 숲 표시
- 생명체 호버 → 말풍선: 어느 동 / 날짜 / 최종 단계
- 지면 텍스처: 생명체 밀도에 따라 황무지 → 풀밭 → 숲
- 특정 지역 클릭 → 해당 시/구로 줌인

## 픽셀 모드 (줌인)

- 읍/면/동 단위 픽셀 표시
- 집중 중인 유저 있는 동네 → 초록 명도로 활성도 표현
- 아무도 없는 동네 → 회색
- 호버 (유저 있음) → 말풍선: 닉네임, 할 일, 남은 타이머
- 호버 (유저 없음) → 말풍선 없음
- 클릭 → 패널 엿보기 모드 전환 + 맵 해당 동네 포커스

## 공통 인터랙션

- 마우스 휠 / 트랙패드 핀치 → 숲 모드 ↔ 픽셀 모드 자연스럽게 전환
- 드래그로 이동 (모바일 터치 드래그 동일)
- 우측 하단 "내 위치" 버튼 고정 → 클릭 시 내 시/구로 이동

## 실시간 구독 항목

- 동네별 활성도 히트맵 (SSE 또는 WebSocket)
- 집중 중인 유저 말풍선 데이터 (호버 시 최신 상태)
