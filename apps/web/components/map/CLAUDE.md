# Map — 맵 시스템 명세 (D)

## 줌 방향 (기획 수정됨)

- **줌인 → 숲 모드** (scale ≥ 1.5, 시/구 단위 크리처 숲 표시)
- **줌아웃 → 픽셀 모드** (scale < 1.5, 전국 동 단위 픽셀 표시)

## 픽셀 모드 (줌아웃)

- 읍/면/동 단위 픽셀 표시 (1픽셀 = 1동)
- 집중 중인 유저 있는 동네 → 초록 명도로 활성도 표현
- 아무도 없는 동네 → 회색 (#707972)
- **호버 (1초 유지 시)** → 툴팁: 동 이름, 오늘의 생명체 단계 + 물주기 현황, 집중 중인 인원수
- 호버 중 마우스 이동 시 포지션 갱신 (타이머 리셋 없음)
- **클릭** → `focusDong(code)` 호출 → 패널 엿보기 모드 전환

## 숲 모드 (줌인)

- 동 픽셀을 숲 팔레트(짙은 초록 계열)로 렌더링
- **호버 가능 영역 단위: 시/구** (dongCode 앞 5자리로 그룹핑)
- 호버 중인 시/구 → 반투명 밝은 초록 오버레이로 하이라이트
- **호버 (1초 유지 시)** → 툴팁: 지역 이름, 오늘의 생명체 단계, 물주기, 집중 중인 전체 인원수
- **클릭** → 해당 동 `focusDong(code)` 호출
- `dongCode` prop으로 포커스 시/구 필터링 (없으면 전체 표시)

## 공통 인터랙션

- 마우스 휠 / 트랙패드 핀치 → 픽셀 모드 ↔ 숲 모드 자연스럽게 전환 (threshold: scale 1.5)
- 드래그로 이동 (모바일 터치 드래그 동일)
- 우측 하단 "내 위치" 버튼 → 클릭 시 전체 보기로 리셋

## 오늘의 생명체 현황 (툴팁 표시 항목)

| 항목 | 내용 |
|---|---|
| 단계 | 🌱 씨앗 / 🌿 새싹 / 🌳 풀 / 🌲 나무 |
| 물주기 | 💧 (완료) / 🩶 (미완료), 최대 3회 |
| 집중 인원 | n명 집중 중 (0명이면 미표시) |
